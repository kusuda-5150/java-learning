# java-learning

『スッキリわかるJava入門』を使ったJava基礎学習用リポジトリです。

## 目的

Javaの基本文法を学習し、最終的にPythonで作成済みの
beatoraja向け統計ツールをJavaで書き直すための土台を作ることが目的です。

現時点では、成果物というよりも学習中の練習コード・確認コードが中心です。

## 実行環境

- Java 21
- IntelliJ IDEA
- Windows

## リポジトリ構成

対応する章ごとにディレクトリを分けて、練習コードと学習メモを置いています。

## 学習方針

既習範囲をもとに、最終目標の統計ツールに応用できる形の練習問題を
LLMに生成させ、解答コードを自分で書いて残しています。

各章の終わりに学習メモを残し、後から振り返られるようにしています。

## 今後の予定

- Javaの基本文法を一通り学習する
- クラスとオブジェクト指向を理解する
- ファイル読み込みやデータ集計処理を学ぶ
- beatoraja統計ツールのJava版を作成する
